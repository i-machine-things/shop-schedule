One-Line Install

Run this on each display device. It installs a kiosk browser service that starts automatically on boot and reconnects after power outages.

Requires Raspberry Pi OS or Debian with a desktop environment (LXDE, GNOME, etc.).

Download script

What It Does

  • Installs Chromium and unclutter
  • Writes /usr/local/bin/shop-kiosk — a launcher that polls this server until reachable, then opens the kiosk in full-screen
  • Creates and enables shop-kiosk.service — auto-starts on boot, restarts automatically on crash or power cycle
  • Disables screen blanking via raspi-config
  • Hides the mouse cursor when idle

Manage

Statussudo systemctl status shop-kiosk
Restartsudo systemctl restart shop-kiosk
Logsjournalctl -u shop-kiosk -f
Removesudo systemctl disable --now shop-kiosk && sudo rm /etc/systemd/system/shop-kiosk.service /usr/local/bin/shop-kiosk
